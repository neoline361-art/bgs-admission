HOW IT WORKS
══════════════════════════════════════════════════════════════════

SAME QR CODE FOR EVERYONE.

Student scans QR → Opens website → Clicks [Apply] → Fills form → Submits → Stored in database forever.

Staff scans same QR → Opens website → Clicks [Staff Login] → Enters username + PIN + CAPTCHA → Logs in → Sees all applications → Manages everything.
