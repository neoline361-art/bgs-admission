# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.x     | ✅ |

## Data Handling

- Supabase anon key is read-only with RLS enforced
- No secrets or API keys are hardcoded
- All authentication is handled server-side via Supabase

## Reporting a Vulnerability

Report privately to: **neoline361@gmail.com**

Do not file public issues for security vulnerabilities.

## Disclosure

1. Report received — acknowledgment within 48 hours
2. Fix developed — target 7 days
3. Patch released with advisory
